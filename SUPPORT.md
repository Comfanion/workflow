# Support

This project is a multi-harness AI agent skills library maintained as a
community effort. To keep the signal high, please route your question to the
right channel.

| I want to… | Go to |
|------------|-------|
| Report a bug (something broken or behaving worse than documented) | [Open an issue — bug report](https://github.com/Comfanion/workflow/issues/new?template=bug_report.yml) |
| Propose a new skill / role / flow change | [Open an issue — feature request](https://github.com/Comfanion/workflow/issues/new?template=feature_request.yml) |
| Ask a "how does X work" question | [GitHub Discussions](https://github.com/Comfanion/workflow/discussions) *(if enabled)* or an issue tagged `question` |
| Report a security issue | **Do not open an issue** — see [SECURITY.md](./SECURITY.md) (private advisory) |
| Contribute code / skills / docs | [CONTRIBUTING.md](./CONTRIBUTING.md) |

## Before you ask

A surprising number of questions are answered in:

- [README.md](./README.md) — overview, install, layout
- [ARCHITECTURE.md](./ARCHITECTURE.md) — the two-layer model + full skill/role catalog
- [FLOW.yaml](./FLOW.yaml) — the recommended pipeline and its hard gates
- The relevant `skills/<name>/SKILL.md` — most procedures are self-documenting

## Response expectations

This is maintained in addition to a day job. Expect best-effort response,
not an SLA — security reports (see [SECURITY.md](./SECURITY.md)) are the one
exception and have stated targets.
